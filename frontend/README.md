# Weather App – Frontend

## Overview

React app for searching and displaying weather information. Features:

- Enter location (city, zip, GPS, etc.)
- Get current weather and 5-day forecast
- Responsive design
- Error handling for failed API requests or invalid locations

## Setup

1. `cd frontend`
2. `npm install`
3. `npm start`

## Requirements

- Node.js
- npm or yarn

---

This app communicates with the backend API (see `/backend`).
