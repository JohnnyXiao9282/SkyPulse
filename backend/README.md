# Weather App – Backend

## Overview

Node.js/Express API server for weather data. Features:

- RESTful API for CRUD operations (weather/location)
- Integrates with weather APIs
- MongoDB for data persistence
- Data export (JSON/CSV)
- Additional API integrations (YouTube, Google Maps)

## Setup

1. `cd backend`
2. `npm install`
3. Create a `.env` file (see `.env.example`)
4. `npm start`

## Requirements

- Node.js
- npm or yarn
- MongoDB (local or cloud)
