# Weather App (Full Stack)

## Overview

A full stack weather app with a React frontend and Node.js/Express backend. Users can search for weather by location, view current weather and a 5-day forecast, and manage weather/location data with CRUD operations. Data is persisted in MongoDB. The backend also supports data export and additional API integrations (e.g., YouTube, Google Maps).

## Structure

- `/frontend` – React app (user interface)
- `/backend` – Node.js/Express API server

## How to Run

See the README files in each subfolder for setup and usage instructions.

## Requirements

- Node.js (v18+ recommended)
- npm or yarn
- MongoDB (local or cloud)
